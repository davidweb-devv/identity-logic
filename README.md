# Who Are You? // Identity Logic

A minimalist, mobile-first web experience that uses recursive logic trees to audit personal focus, clarity, and decision-making patterns.

## The Concept
In 2026, the primary challenge isn't a lack of information—it's a lack of signal. This project is a digital mirror designed to help users identify whether their daily choices are leading toward **Clarity** or **Fragility**.

## Features
- **Recursive Logic Engine:** A dynamic JavaScript-based decision tree that adapts based on user input.
- **High-End Minimalist UI:** A dark-mode, "editorial" aesthetic optimized for mobile performance.
- **Identity Analysis:** A post-sequence audit that calculates a "Logic Persona" based on weighted risk/reward variables.

## Tech Stack
- **HTML5** (Semantic structure)
- **CSS3** (Hardware-accelerated transitions & custom typography)
- **Vanilla JavaScript** (State management and logic processing)

## Developmental Note
This project was built entirely on a mobile device, emphasizing high-efficiency code and a "less is more" design philosophy.
